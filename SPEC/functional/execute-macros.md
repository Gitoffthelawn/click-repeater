# РЕЖИМ ИСПОЛНЕНИЯ MACROS

---

## ЗАПУСК, СТОП

- По клику на иконку расширения в панели браузера:
   - Открывает popup
   - Ничего не запускает
   - Если было запущено, то останавливает режим (возможны доп.требования для разных режимов)
- Запуск режима создания строго по кнопке в popup
- Запуск режима исполнения:
   - По кнопке в popup для конкретного macros
   - По хоткею для дефолтного macros

## ИМИТАЦИЯ ЧЕЛОВЕКА

### Общие правила
- Не прыгать курсором
- Путь в runtime
- Неровный путь
- Микросдвиги
- Рандом: неск. мм
- Рандом точки
- Сдвиг клика: 1 mm
- Заполнять movementX
- Заполнять movementY
- Паузы в runtime
- Не повторять тайминг
- Избегать ровный темп

### Порядок действий

Условия цикла:
- Пока есть следующий шаг
- И нет клика пользователя
- И не нажата остановка

Цикл:
1. Подготовить цель
    1. Найти элемент или точку. Если цель не найдена: стоп
    2. Задать точку клика
    3. Сдвиг клика: 1 mm
2. Если нужен путь
    1. Рассчитать путь в runtime
    2. Учесть дистанцию
    3. Добавить микросдвиги
    4. Рандом: неск. мм
    5. Послать pointermove
    6. Послать mousemove
    7. Заполнить movementX/Y
3. Если цель новая
    1. Послать pointerover
    2. Послать pointerenter
    3. Послать mouseover
    4. Послать mouseenter
    5. Пауза: 80-250 ms
4. Выполнить клик
    1. Послать pointerdown
    2. Послать mousedown
    3. Удержать 50-150 ms
    4. Послать pointerup
    5. Послать mouseup
    6. Пауза: 20-120 ms
    7. Послать click
5. Завершить шаг
    1. Пауза: 500-1000 ms
    2. Не повторять тайминг
    3. Перейти к шагу дальше
6. Запустить цикл для следующего клика, если:
    - Есть следующий шаг
    - Пользователь не кликнул
    - Стоп не нажат
    - Цель найдена
