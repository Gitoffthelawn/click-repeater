# `icons/lucide`

Local copy of the Lucide SVG icons used by Macros Repeater.

- Vendor code: SVG path data and icon markup from Lucide.
- Local adaptation: icons are exposed through `icons.js` for non-module extension scripts; tracker icons use percentage dimensions so they scale with the cursor marker.
- License: `LICENSE`.
- Runtime API: `globalThis.macrosRepeaterLucideIcons`.
